# E-Commerce Back End Database

A back-end program for E-commerce.

## Description

A Node.js Express server that interacts with a SQL database to store info about products and what categories they belong to.

## Getting Started

### Dependencies

* Node.js
* npm
* Express.js
* MySQL2
* Sequelize
* dotenv
